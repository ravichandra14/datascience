# 🚀 Data Analytics and Machine Learning Portfolio

Welcome to my comprehensive practice repository for **Data Analytics** and **Machine Learning**! This repository serves as a centralized portfolio of various data projects, ML algorithms, exploratory analyses, and coding experiments built using industry-standard Python libraries.

## 🎯 About This Repository

This repository documents my journey and practice in data science. It covers everything from fundamental data manipulation to end-to-end machine learning modeling. Features include:
- **Exploratory Data Analysis (EDA)** on diverse, real-world datasets.
- **Machine Learning Algorithms** organized by categorization (Classification, Clustering, Regression).
- **End-to-End ML Projects** solving practical business use-cases.
- **Data Preprocessing Pipelines** for cleaning and scaling raw datasets.

## 🛠️ Tech Stack

- **Language:** Python
- **Data Manipulation:** NumPy, Pandas
- **Machine Learning:** Scikit-Learn
- **Data Visualization:** Matplotlib, Seaborn
- **Environment:** Jupyter Notebooks

## 📂 Directory Structure

- **`data-analytics/`**: Contains core EDA projects (COVID-19, IPL, Netflix, Sales) and fundamental library practice (NumPy arrays/broadcasting, Pandas mini-projects, Matplotlib/Seaborn visualizations).
- **`machine-learning/`**: 
  - *Classification*: Decision Trees, KNN, Naive Bayes, Random Forest, SVM.
  - *Clustering*: Hierarchical, K-Means.
  - *Regression*: Linear, Logistic.
  - *ML Projects*: Full projects like Customer Churn/Segmentation and House Price Prediction.
- **`datasets/`**: Segregated by pipeline stage (`raw/`, `cleaned/`, `processed/`) for various datasets like Iris, Titanic, Mall Customers, SMS Spam, and more.
- **`notebooks/`**: Scratchpad Jupyter notebooks for quick experiments, tutorials, and practice coding sessions.
- **`utilities/`**: Modularized helper Python scripts for reusable data preprocessing and visualization functions.

## 📊 Highlighted Projects

### 1. Exploratory Data Analysis
- **COVID-19 Analysis:** Tracking trends, recovery rates, and visualizing the spread of the virus over time.
- **IPL Analysis:** Detailed insights into team performances, player statistics, and match outcomes.
- **Netflix Analysis:** Exploring the streaming platform's content library, genres, runtime, and release trends.
- **Sales Analysis:** Understanding revenue streams, profit margins, and customer purchasing behaviors.

### 2. Machine Learning Projects
- **Customer Segmentation System:** Utilizes `K-Means clustering` to group mall customers based on their Annual Income and Spending Score, helping identify target demographics.
- **House Price Prediction:** A robust regression pipeline to predict property values based on various housing features.
- **Spam Detection:** A `Naive Bayes` classification project to filter standard SMS messages from spam.

## ⚙️ Setup and Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/ravichandra14/datascience.git
   cd datascience
   ```

2. **Create a Virtual Environment (Recommended)**
   ```bash
   python -m venv venv
   # On Windows:
   venv\Scripts\activate
   # On macOS/Linux:
   source venv/bin/activate
   ```

3. **Install Dependencies**
   Make sure you have your requirements ready, then install them:
   ```bash
   pip install -r requirements.txt
   ```

4. **Launch Jupyter Notebook**
   ```bash
   jupyter notebook
   ```
   Navigate to the respective folders (e.g., `machine-learning/ml-projects/`) and open the `.ipynb` files to view the code and models.
